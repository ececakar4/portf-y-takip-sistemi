Proje Özellikleri
Dinamik Veri Girişi: Kullanıcıdan input fonksiyonu ile varlık adı, fiyat, adet bilgilerini alır.
Hata Yönetimi: try-except bloğu ile fiyat ve adet girişlerinde hatalı verileri ayıklar.
Akıllı Çıkış Mekanizması: 'q' tuşuna basıldığında döngüden güvenli bir şekilde çıkar.

Pandas kütüphanesi kullanılmıştır.

KULLANIM KILAVUZU
Projeyi çalıştırdığınızda karşınıza gelen ekrana varlık ismini girin.
Varlığın birim fiyatını ve elinizdeki adedini giriniz.
Başka varlık eklemek istemediğinizde 'q' harfine basarak sonlandırın ve sonuç tablosunu inceleyin.
